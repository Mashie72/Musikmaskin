# 🎵 Musikmaskin – Build & Release Instructions

Version: v1.0.2
Datum: 2025-06-25

---

## 🧱 Allmänt

För att bygga binärer behöver du:

- Python 3.11 eller 3.12 (rek. samma som projektets venv)
- Virtuell miljö (venv)
- PyInstaller (`pip install pyinstaller`)
- Projektfiler: `musikmaskin.py`, `requirements.txt`, ev. `.ico`, `README.md`, etc.
- (Mac) Använd Homebrew för dependencies
- (WSL/Linux) Kontrollera att `ffmpeg`, `python3-tk`, `soundfile` etc är installerat
